# Pokemon Memory Card Game

A dynamic memory card game built as part of **The Odin Project** curriculum.  
The goal is simple: click each card **only once**. Clicking the same card twice resets your score.

This project focuses on **state management**, **component re-renders**, and **React fundamentals**.
